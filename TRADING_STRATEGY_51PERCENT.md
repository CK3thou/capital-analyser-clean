| **Profit Target 2** | +4% gain | Sell another 30% (consolidate gains) |
| **Profit Target 3** | +6%+ gain | Sell remaining 40% (close position) |
| **Trailing Stop Loss** | 2% from entry | Automatic exit (risk management) |
| **RSI Divergence** | Price ↑ but RSI ↓ | Exit 50% immediately |
| **Support Break** | Price breaks support | Exit remaining position |
| **Hold Duration** | >5 trading days | Reassess momentum signal |

### **D. STOP LOSS RULES (Risk Management)**

- **Hard Stop Loss:** -2% from entry price (MANDATORY)
- **Reason:** Protects against gap downs and rapid reversals
- **Position Size Calculation:** 
  - Max loss per trade = 1-2% of account
  - If account = $10,000 → Max loss = $100-200
  - Position size = Max Loss / Stop Distance
  - Example: $200 loss / 2% = $10,000 position size
